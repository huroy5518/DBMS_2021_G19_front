# DBMS_G19_Front_End
---
## In order to run this project, you need to set the nodejs version to 14
### You may setup nodejs version with nvm
```
nvm install 14
nvm use 14
```
### Usage
```
git clone https://github.com/huroy5518/DBMS_2021_G19_front.git
cd DBMS_2021_G19_front
npm install
npm start
```
